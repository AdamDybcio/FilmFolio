## Description
So, this application is used to search and browse movies. There is no account creation, because movies bookmarked by the user are saved locally and an external database is not needed.</br></br>

The only feature I haven't added is multi-language support. I decided that it's not needed at the moment, although this feature is almost fully implemented (all you need to do is add the feature to the button in the settings so that queries return data in the appropriate language and translate texts like headers accordingly. </br></br>

I don't know if there's anything more to write here, it's best to check for yourself how this application works and what functions it has through a simple installation! üòÅ

## Installation
### 1. Clone (or Fork) the Repository
```
git clone https://github.com/AdamDybcio/FilmFolio.git
cd FilmFolio
```

### 2. Run pub get
```
flutter pub get
```

### 3. Run the app
```
flutter run
```

### (Important!!! After you run the app, go to the settings page and enter your TMDB API KEY, otherwise you will not be able to load any data! (You can get the API KEY here -> https://www.themoviedb.org/settings/api)
